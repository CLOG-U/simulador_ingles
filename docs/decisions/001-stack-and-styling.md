# 001 - Stack y estilos

**Fecha:** 2026-07-16

## Decisiones

- **CSS:** Tailwind CSS v4 para el frontend (utilidades + tokens de marca en `@theme`).
- **Estado remoto:** TanStack Query (Fase 4).
- **Formularios:** React Hook Form + Zod (Fase 4).
- **Hash de contraseñas:** Argon2id via `argon2-cffi` (Fase 2).
- **IDs públicos:** UUID v4 en todas las tablas.
- **Fechas:** UTC en base de datos.

## Desambiguaciones del banco (sección 4.4 del plan)

Aplicadas en el seed. Variantes británicas (`learnt`, etc.) **no** incluidas en el MVP.

## Logo

Se usa la paleta del plan (sección 8). El activo final PNG/SVG queda pendiente del cliente.
